# 🚀 The Project - Department Review Setup

**Quick Setup for Non-Technical Department Reviews**

> This is a **demo environment** - no real cryptocurrency or wallets needed!

## ⚡ Quick Start (3 minutes)

### 1. Prerequisites

- **Node.js 18+**: Download from [nodejs.org](https://nodejs.org/)
- **Modern Browser**: Chrome, Firefox, Safari, or Edge

### 2. One-Click Setup

```bash
# Run the automated setup script
./scripts/setup-review.sh
```

### 3. Start Reviewing

```bash
# Navigate to the project directory
cd the-project

# Start the application
npm run review:start
```

Your browser will open to `http://localhost:3000`

## 🆘 Need Help?

### Quick Fixes

```bash
# If something goes wrong
cd the-project
npm run clean
npm install
npm run review:start
```

### Common Issues

- **Blank page**: Clear browser cache (Ctrl+F5)
- **Won't start**: Check Node.js version (node -v)
- **Slow loading**: Close other browser tabs
